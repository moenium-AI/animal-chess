# 🐻 どうぶつチェス (Animal Chess)

かわいい動物のピースで遊べるブラウザチェスゲームです。インストール不要、ブラウザだけで遊べます。

## 🎮 あそぶ

👉 **https://moenium-ai.github.io/animal-chess/**

上のリンクを開くだけ。スマホでもパソコンでも遊べます。

## ✨ 機能

- **3Dモード(ドット絵)** — SFC時代のシミュレーションRPG風に盤へ角度をつけ、
  擬人化どうぶつのドット絵キャラが待機モーションつきで立ちます
  - 🐤ポーン=ひよこ / 🐴ナイト=うま / 🦉ビショップ=ふくろう / 🐻ルーク=くま / 🐱クイーン=ねこ / 🦁キング=ライオン
  - 取られたコマはドット絵の馬車で相手チームの**宴会会場**へ。
    食べたり🍖飲んだり🍺寝転んだり💤、リアルタイムで気ままに宴会を楽しみます
- **シンプルモード** — 絵文字ベースのピースに切り替え(クラシック+動物セット4種類)
- **コマ種類バッジ** — どのコマか一目でわかる記号(♟♞♝♜♛♚)を隅に表示(ON/OFF可)
- **🎵 BGM** — やさしいオリジナル曲を全10曲収録。「オート」で順番に流すか、好きな1曲をずっと流すかを選べます(ON/OFF可)
- **コンピューター対戦** — 強さレベル1〜5
- **ふたりで遊ぶ** — 同じ画面で交互に操作
- **💡 ヒント機能** — 次の一手をAIがアドバイス
- **📜 棋譜(PGN)読み込み** — 貼り付け/ファイル読み込み、再生、途中局面から対局再開
- **🧩 チェスパズル** — 1手詰め・2手詰め全16問

外部への通信は一切ありません。設定とパズルのクリア状況はブラウザ内(localStorage)に保存されます。
